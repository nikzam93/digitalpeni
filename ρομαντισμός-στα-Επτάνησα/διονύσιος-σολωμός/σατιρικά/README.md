# Κώνστα Λ.Γ.Χ. (επιμ.), *Διονυσίου Σολωμού Άπαντα. Τα Ευρισκόμενα*, Αθήνα: Τύποις Π.Δ. Σακελλαρίου, 1901.

##**Ἡ Πρωτοχρονιά**

Ἄ! μὲ τάϊζε κουκουνάρια! / Θέλει τὸ χετε ἀκουστά. (σ. 227)

##**Τὸ ἰατροσυμβούλιο**

Μωρέ, σὺ σαὶ ἕνα λουλούδι / Κ᾽ ἐγὼ ἐγίνηκα κουκκοῦδι. (σ. 232)

(Μὰ νὰ μὴν εἶναι ὁ Σιὸρ πάρες) / Νὰ μοῦ στείλῃ ἐκειὰ τὰ κλήματα· (σ. 233)

Μὲ τὰ κλήματα ᾽ς τὸ νοῦ του, / Πάει ᾽ς τὸ σπίτι τ᾽ ἀνεψιοῦ του, [...] (σ. 233)
